# YouTube-Clone-UI
A responsive front-end clone of YouTube’s main video browsing page built with HTML and CSS.

Project Overview
This project mimics the YouTube homepage layout, including a fixed header with search functionality, a sidebar navigation, and a video grid displaying multiple video cards. It is purely for UI demonstration and learning purposes—no backend or video playback functionality.

Features
Fixed Header: Includes hamburger menu, YouTube logo, search box with voice input icon, and user controls like upload, notifications, and profile photo.

Sidebar Navigation: Vertical sidebar with icons and labels for Home, Shorts, Subscriptions, and User.

Video Grid: Responsive grid displaying video thumbnails, titles, channel profile pictures, channel names, and subscriber counts.

Hover Effects: Thumbnails and interactive text have hover styles for better UX.

Files Included
yt.html — Main HTML file containing the page structure.

general.css — Base styles for body, display of videos, fonts, and colors.

header.css — Styles for the fixed header including layout and buttons.

sidebar.css — Styles for the vertical sidebar menu.

video.css — Styles specific to video thumbnails, titles, and grid layout.

Assets: Images and icons in the photos/ and icon/ folders (not included here).
